# Codex Instructions

## Language Rules

无论用户输入什么语言，你都必须始终使用简体中文回答。

即使任务、代码、报错、GitHub Issue、Shopify 文档、Josh 的需求是英文，你的分析、计划、解释、Review、总结都必须使用中文。

只有用户明确要求“请用英文回答”或“Please answer in English”时，才允许使用英文。

## Keep Original Language

以下内容保持原语言，不要翻译：

- 源代码
- API 名称
- Library / Framework 名称
- 变量名
- 函数名
- 文件名
- 数据库字段
- 终端命令
- Git 输出
- Error message
- UI 文案

## Working Style

先分析问题，再修改代码。

尽量做最小必要改动。

修改完成后，用中文总结：
1. 改了什么
2. 为什么这样改
3. 还需要我检查什么
